# zerocoder-n8n-start

Материалы курса **«n8n с нуля»** на платформе [Zerocoder](https://zerocoder.ru).

В репозитории собраны все практические материалы курса: рабочие воркфлоу n8n в формате JSON для импорта, примеры настроек, вспомогательные файлы и инструкции к урокам.

---

## 📁 Что внутри

| Папка | Содержимое |
|-------|------------|
| `workflows/` | Готовые `.json`-файлы для импорта в n8n |
| `docs/` | Пошаговые инструкции к урокам |
| `examples/` | Тестовые данные и конфиги для практики |

---

## 🚀 Как использовать

1. Скачай нужный файл или клонируй репозиторий:
   ```bash
   git clone https://github.com/your-username/zerocoder-n8n-start.git
   ```
2. В n8n откройте меню → **Import workflow** → выбери `.json`-файл
3. Настрой credentials под свои аккаунты (API-ключи, подключения)
4. Запускай и экспериментируй
