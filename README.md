Материалы курса «n8n с нуля» на платформе Zerocoder.
В репозитории собраны все практические материалы курса: рабочие воркфлоу n8n в формате JSON для импорта, примеры настроек, вспомогательные файлы и инструкции к урокам.
Что внутри

📦 Воркфлоу — готовые .json-файлы для импорта в n8n
📝 Инструкции — пошаговые гайды к каждому уроку
🗂️ Примеры — тестовые данные и конфиги для практики

Как использовать

Скачай нужный файл или клонируй репозиторий
В n8n откройте меню → Import workflow → выбери .json-файл
Настрой credentials под свои аккаунты (API-ключи, подключения)
Запускай и экспериментируй
